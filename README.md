# Alex V. Nunez — Resume Website

Personal resume website showcasing engineering background, technical expertise, and project experience across automation, control systems, software development, and IT support. The site includes academic achievements, published research, bilingual communication skills, and examples of projects and tools.

## Features
- Clean, responsive resume website built with Bootstrap templates and custom styling
- Sections for experience, education, projects, publications, and contact
- Mobile-first layout and printable resume view

## Technologies
- HTML5, CSS3
- Bootstrap (local or CDN)
- Small JavaScript for interactivity (optional)

## Preview (Windows)
Open `index.html` directly in your browser.

## Project structure (example)
- `index.html` — main resume page
- `styles.css`, `blog.rtl.css` — custom styles
- `assets/` — images, icons, Bootstrap dist files
- `js/` — optional scripts
